# OGame Clone Server

This is the server-side implementation of the OGame clone project built with Node.js and Express. The server handles authentication, game logic, and data management for the client-side application.

## Project Structure

- **src/**: Contains the source code for the server.
  - **routes/**: Defines the API routes for authentication and game management.
  - **controllers/**: Contains the logic for handling requests and responses.
  - **models/**: Defines the data models for users, planets, and buildings.
  - **middleware/**: Contains middleware functions for authentication and other purposes.
  - **server.js**: The entry point for the Node.js server.

## Getting Started

1. **Clone the repository**:
   ```
   git clone https://github.com/yourusername/ogame-clone.git
   cd ogame-clone/server
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the server**:
   ```
   npm start
   ```

4. **API Documentation**: Refer to the routes defined in the `routes` directory for available endpoints and their usage.
