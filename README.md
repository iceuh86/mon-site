# OGame Clone

This project is a web-based game inspired by OGame, where players can build their own space empire, manage resources, and engage in strategic gameplay.

## Project Structure

```
ogame-clone
├── src
│   ├── index.html        # Main HTML file
│   ├── css
│   │   ├── style.css     # General styles
│   │   └── game.css      # Game-specific styles
│   ├── js
│   │   ├── app.js        # Application initialization and state management
│   │   ├── game.js       # Core game logic
│   │   ├── player.js     # Player class and management
│   │   └── utils.js      # Utility functions
│   └── assets
│       └── data.json     # Game data in JSON format
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Features

- Build and manage your own space empire.
- Resource management and strategic gameplay.
- Engaging user interface with responsive design.
- Player management system with customizable attributes.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/ogame-clone.git
   ```

2. Navigate to the project directory:
   ```
   cd ogame-clone
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to start playing.
